# Contributing to PostCare AI

Thank you for your interest in contributing to PostCare AI! We welcome contributions from the community to help improve healthcare follow-up automation.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Follow the Setup & Installation** guide in the [README.md](README.md).
3. **Create a new branch** for your feature or bug fix: `git checkout -b my-new-feature`.
4. **Make your changes** and ensure the code follows our style guidelines.
5. **Commit your changes**: `git commit -m "feat: descriptive commit message"`.
6. **Push to your branch**: `git push origin my-new-feature`.
7. **Create a Pull Request** against the `master` branch.

## Development Workflow

-   **Frontend**: React + Vite in `client/`
-   **Backend**: Express.js in `server/`
-   **Mobile**: Expo/React Native in `mobile/` (separate repo for now)

## Code Standards

-   Use clean, readable code.
-   Add comments to complex logic.
-   Ensure API responses follow the established `{ success: true, data: ... }` pattern.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository.
